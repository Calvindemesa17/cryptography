One Time Pad Encryption

The one-time pad cipher, also known as the Vernam cipher, is a cryptographic algorithm that provides perfect secrecy when used correctly. It operates by combining the plaintext with a random key stream of the same length. This key stream is typically generated by a truly random process or algorithm, and it should only be used once, hence the name "one-time pad."
